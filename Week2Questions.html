Week #1 Questions and Answers

HTML Question

In HTML, Document Object Model(DOM) works similar to which Data Structure?

DOM treats HTML document as a tree. Each node in tree is an object representing a part of HTML document. For example, HTML Tag is a node and it has head tag and body tag as the children nodes.

CSS Question

Which CSS property can cause overlapping in HTML elements?

Negative margins
Example: margin-top:-200px

Website Structure Question 1

What is the parameter of a good website navigation and structure?

User should be able to navigate wherever they want in minimum number of clicks. It should not be complicated to navigate from one page to another.

Website Structure Question 2

How one can optimize a website? Mention some ways.

  • Avoid loading unused CSS modules or scripts
  • Use responsive images and load images according to user's case
  • User server side caching mechanisms
